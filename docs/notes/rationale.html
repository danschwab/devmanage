Rationale:

Top Shelf Exhibits currently has no consolidated inventory tracking whatsoever. This system would be an in-house solution that unifies several inventory touchpoints that are currently disconnected.

Possibilities:

  • Generate initial pack lists directly from Inventor models as early as concept stage
  • Identify item shortages as pack lists are generated (including beMatrix items if added)
  • Enable tracking of item conditions and improve efficiency of repair schedule
  • Provide historical data analysis to improve busy season planning
  • Quick inventory or item history checks and updates via qrcode scanning
  • Easy cost tracking, pricing, and budgeting analytics if internal pricing fields are added
  • Mobile-friendly for on-the-go inventory management and status checks
